Características Principales

Interfaz Web

Accede desde cualquier dispositivo en tu red local con un navegador web moderno.

Acceso Seguro

Autenticación por DNI y contraseña para garantizar la seguridad de los registros.

Base de Datos SQL

Almacenamiento confiable y organizado de todos los registros de asistencia.

¿Cómo Funciona?

1

Registro de Empleados

Los empleados se registran una sola vez con su DNI y contraseña.

2

Registro de Entrada/Salida

Los empleados inician sesión para registrar su entrada o salida con un solo clic.

3

Consulta de registros

Los trabajadores pueden consultar todos sus registros de asistencia usando sus credenciales.

Requisitos del Sistema

  • Windows

    Sistema operativo recomendado (Linux disponible a través de Wine sin soporte oficial)

  • Java 21

    Recomendamos usar la versión de Eclipse Adoptium

  • MySQL

    Para el almacenamiento de datos
